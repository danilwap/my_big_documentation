Контейнеры поднимаются из образа, есть 3 способа получить образа
- Команда docker pull <Образ> - скачать из репозитория (Docker Hub)
- На основе готового образа создать свой через Dockerfile
- Команда docker commit <Контейнер> - Создать образ на основе контейнера

docker images 
docker ps
docker ps -a

docker pull <Образ> - скачать образ
docker pull <Образ>:16 - скачать образ определённой версии

docker rmi <Образ>:<tag> - удалить образ определённой версии
