1) Открыть командную строку с правами администратора
2) В нем вам нужно ввести команду wsl --install
3) перезагрузить компьютер
4) Когда перезагрузитесь, опять откройте командную строку в режиме администратора. 
Затем выполните следующую команду:  wsl --install -d Ubuntu
Вам нужно создать имя и пароль.
5) Перейдите на сайт докера, где предлагают установить его на винду https://docs.docker.com/desktop/install/windows-install/
Ознакомьтесь с ним и скачайте установщик Docker Desktop for Windows (файл .exe).
Когда он скачается, нажмите на этот установщик. Обратите внимание, что у вас должна стоять галочка на Use WSL 2 instead of Hyper-V. 

wsl - запускает ubuntu
docker -v
docker-compose -v

В курсе нам нужно будет прокидывать файлы в докер-контейнеры, а также использовать файлы для докер компоуза.
Чтобы попасть в нужную папку, можно в вашей убунте выполнить команду explorer.exe .
