JOIN - объединение нескольких таблиц в одну

В SQL существуют следующие типы соединений:

INNER JOIN — возвращает только те строки, которые имеют соответствующие значения в обеих таблицах, участвующих в объединении.

LEFT JOIN — возвращает все строки из левой таблицы и только соответствующие строки из правой таблицы.

RIGHT JOIN — возвращает все строки из правой таблицы и только соответствующие строки из левой таблицы.

FULL JOIN (или FULL OUTER JOIN) — возвращает все строки из обеих таблиц, даже если нет соответствующих строк в другой таблице.

CROSS JOIN — возвращает декартово произведение двух таблиц, то есть все возможные комбинации строк из обеих таблиц.


SELECT pp.*, p.* FROM product as pp
LEFT JOIN product_photo as p
ON pp.id = p.product_id
